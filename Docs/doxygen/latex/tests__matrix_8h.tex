Регистрирует все тестовые случаи для операций с матрицами 

Данная функция создает тестовый набор для операций с матрицами и добавляет в него все индивидуальные тестовые случаи. Тесты включают\+:
\begin{DoxyItemize}
\item Создание матриц и управление памятью
\item Операции ввода/вывода с файлами
\item Базовые арифметические операции
\item Свойства матриц и преобразования
\item Особые случаи и граничные условия
\end{DoxyItemize}

\begin{DoxyNote}{Заметки}
Эту функцию необходимо вызывать перед запуском любых тестов матриц 
\end{DoxyNote}
