Выводит матрицу с пользовательским форматированием

Аргументы
matУказатель на матрицу для вывода
formatСтрока формата printf для элементов
Заметки
Примеры форматов:
  • "%8.3f " - выравнивание по 8 символам, 3 знака после точки
  • "%+.2e " - экспоненциальная запись со знаком
Предупреждения
Не проверяет валидность строки формата

◆ save_matrix_to_file()

int save_matrix_to_file ( const Matrix *  mat,
const char *  filename 
)

Сохраняет матрицу в файл

Аргументы
matУказатель на матрицу для сохранения
filenameИмя файла для сохранения
Возвращает
0 при успешном сохранении, -1 при ошибке
Заметки
Формат файла:
  • Первая строка: количество строк и столбцов
  • Последующие строки: элементы матрицы
Предупреждения
При ошибке открытия файла возвращает -1 и выводит сообщение в stderr
Аргументы
matУказатель на матрицу для сохранения
filenameИмя файла для сохранения
Возвращает
0 в случае успеха, -1 при ошибке
Заметки
Формат файла:
  • Первая строка: количество строк и столбцов
  • Последующие строки: элементы матрицы
  • Элементы сохраняются с фиксированной точностью 6 знаков после запятой
