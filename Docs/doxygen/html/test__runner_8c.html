◆ main()

int main ( )

Главная функция, которая инициализирует фреймворк тестирования CUnit, регистрирует тестовые наборы и запускает тесты.

Эта функция инициализирует реестр тестов CUnit, регистрирует тесты операций с матрицами и вывода, устанавливает основной режим работы в подробный, выполняет тесты и очищает реестр.

Возвращает
Возвращает 0 при успехе или ненулевой код ошибки, если инициализация или выполнение тестов не удались.
